# Algorithms

## Extraction Algorithms

The toolkit currently implements several analytical parameter extraction algorithms.

### Threshold Voltage

Extract threshold voltage from transfer characteristics.

---

### Mobility

Estimate carrier mobility using strong inversion equations.

---

### Channel-Length Modulation

Determine λ from output characteristics.

---

### Output Resistance

Estimate output resistance from saturation-region fitting.

---

### Optimization

Support iterative optimization and multi-bias regression for improved robustness.
