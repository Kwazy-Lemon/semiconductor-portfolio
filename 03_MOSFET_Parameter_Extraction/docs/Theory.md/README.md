# Theory

## Overview

The MOSFET Parameter Extraction Toolkit is based on analytical Level-1 MOSFET compact models.

The theoretical foundation includes semiconductor device physics, MOS capacitor theory, and long-channel MOSFET equations.

- Semiconductor fundamentals

- Threshold voltage
- Strong inversion
- Drain current equations
- Carrier mobility
- Channel-length modulation
- Output resistance
- Parameter sensitivity
